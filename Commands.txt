To Run Standalone grid
----------------------
cmd
java -jar selenium-server-4.23.1.jar standalone
http://localhost:4444/ui/#

To Start Grid on Docker
----------------------
cd C:\Users\ADMIN\git\EcommerceV21
docker-compose up


To Remove Containers on Docker
------------------------------
cd C:\Users\ADMIN\git\EcommerceV21
docker-compose down


To Run pom.xml in Command Prompt
-------------------
cd C:\Users\ADMIN\git\EcommerceV21
mvn test
